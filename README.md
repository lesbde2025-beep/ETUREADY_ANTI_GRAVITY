Deployer les modif dans anti gravity vers le site en ligne publique : 

À partir de maintenant, à chaque fois que tu modifies ton site dans Antigravity et que tu veux mettre en ligne, tu n'auras plus qu'à taper ces 3 commandes dans le terminal :


git add .
git commit -m "description de ta modification"
git push


Et Netlify redéploie automatiquement en 1-2 minutes ! 🚀
